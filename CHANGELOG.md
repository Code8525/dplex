# Changelog

## [0.2.0] - 2026-01-04

### ⚠️ Breaking Changes

#### Удален параметр `key_type`

**Что изменилось:**
- Параметр `key_type=uuid.UUID` удален из всех конструкторов и методов
- Тип первичного ключа теперь определяется автоматически через generic параметры классов

### Добавлено

- Утилиты для deprecation warnings (`dplex.internal.deprecation`)
- Руководство по миграции для разработчиков

### Исправлено

- Улучшена типизация для `BaseNumberFilter` и `BaseDateTimeFilter`
- Исправлены ошибки mypy в `filter_applier.py`

## [0.1.0] - 2025-11-01

### Добавлено

- Первый релиз dplex
- Базовая функциональность фильтрации, сортировки и пагинации
- Поддержка 11 типов фильтров
- Repository и Service паттерны
